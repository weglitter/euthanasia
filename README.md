# Global Euthanasia Legalization & Awareness Campaign Website  

This repository is a project to create a **global euthanasia legalization campaign website**.  
It aims to provide accurate information about euthanasia and promote the campaign for legalization.  

## 🌍 Project Goals
- Provide **objective and reliable information** about euthanasia  
- Introduce the legalization processes of euthanasia in various countries  
- Build a platform where campaign participants can share opinions and engage in discussions  

## 📌 Key Features
- **Information Page**: Laws related to euthanasia and historical data on legalization from around the world  
- **Campaign Participation**: Online petitions, discussion boards, and event announcements  
- **Multilingual Support**: Translations to ensure global accessibility  
- **Research Archive**: Collection of papers, articles, interviews, and other reference materials on euthanasia  

## 🚀 Installation & Execution
To run this project on your computer, follow these steps:

1. **Clone the Repository**
   - Open the terminal (command prompt) and enter:
     ```bash
     git clone https://github.com/username/euthanasia-campaign.git
     ```
   - This will copy the project from GitHub to your computer.

2. **Navigate to the Project Folder**
   - Move into the cloned folder:
     ```bash
     cd euthanasia-campaign
     ```

3. **Open the Homepage**
   - Double-click the `index.html` file inside the folder, or  
   - Open the `index.html` file in your browser to view the website.
